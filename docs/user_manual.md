## 系统简介

本地文件AI管理系统是一个基于RAG（检索增强生成）技术的智能文件管理平台，帮助您高效管理和检索本地文档。

### 主要功能

- **文件上传与管理**: 支持多种文件格式（PDF、Word、Excel、TXT等）
- **智能搜索**: 基于语义理解的文档搜索，快速找到相关内容
- **RAG对话**: 与文档进行自然语言对话，获取精准答案
- **文档生成**: AI辅助生成文档大纲和内容
- **自动分类**: 智能识别文档类型并自动分类

### 技术特点

- 完全本地运行，数据安全可控
- 支持中英文混合处理
- 流式响应，实时反馈
- 多模态数据处理能力

## 快速开始

### 系统要求

- Python 3.8+
- 8GB+ 内存
- 10GB+ 可用磁盘空间

### 安装步骤

1. 克隆项目仓库
```bash
git clone <repository-url>
cd 本地文件AI管理系统
```

2. 安装依赖
```bash
pip install -r config/requirements.txt
```

3. 启动服务
```bash
python startall.py
```

4. 访问系统
打开浏览器访问: `http://localhost:9988`

### 首次使用

1. 登录系统（如需要）
2. 上传您的文档
3. 等待文档处理完成
4. 开始使用搜索和对话功能

## 文件管理

### 上传文件

1. 点击"上传文件"按钮
2. 选择要上传的文件
3. 等待文件上传和处理完成

**支持的文件格式**:
- 文档: PDF, DOC, DOCX, TXT, MD, PPT, PPTX, RTF, HTML, HTM
- 表格: XLSX, XLS, CSV
- 其他: JSON, XML

**文件大小限制**: 最大50MB

### 查看文件列表

文件列表页面显示所有已上传的文件，包括：
- 文件名
- 文件类型
- 文件大小
- 上传时间
- 分类标签
- 处理状态

### 文件操作

- **查看详情**: 点击文件名查看详细信息
- **删除文件**: 点击删除按钮移除文件
- **重新处理**: 对处理失败的文件重新进行向量化

### 自动分类

系统会自动识别文档类型并分类：
- 技术文档
- 学术论文
- 商业报告
- 其他

您也可以手动修改分类标签。

## 智能搜索

### 基础搜索

在搜索框中输入您的问题或关键词，系统会返回最相关的文档片段。

**搜索示例**:
- "如何使用RAG技术"
- "机器学习的基本概念"
- "项目开发流程"

### 高级搜索

使用高级搜索功能可以：
- 按分类筛选
- 指定返回结果数量
- 查看相似度分数

### 搜索结果

搜索结果包含：
- 相关文档片段
- 文档来源信息
- 相似度评分
- 上下文预览

点击结果可以跳转到文档详情页面。

## RAG对话

### 开始对话

1. 在对话界面输入您的问题
2. 系统会检索相关文档并生成回答
3. 支持多轮对话，上下文连贯

### 对话技巧

- **具体问题**: 提出具体明确的问题会得到更准确的答案
- **上下文**: 可以引用之前的内容进行追问
- **多文档**: 系统会从多个文档中综合信息

### 对话示例

**用户**: 什么是RAG技术？

**系统**: RAG（Retrieval-Augmented Generation）是一种结合检索和生成的AI技术。它通过检索相关文档内容，然后基于检索到的信息生成回答，可以提高回答的准确性和可靠性。

**用户**: RAG有哪些应用场景？

**系统**: RAG技术主要应用于以下场景：
1. 智能问答系统
2. 文档分析和总结
3. 知识库构建
4. 客服机器人
...

## 文档生成

### 生成大纲

1. 输入文档主题
2. 选择文档风格（学术、技术、商业等）
3. 设置大纲深度
4. 点击生成

### 生成文档

1. 基于生成的大纲
2. 选择内容风格
3. 系统会流式生成文档内容
4. 可以实时查看和编辑

### 文档风格

- **学术**: 严谨、正式，适合论文和报告
- **技术**: 专业、实用，适合技术文档
- **商业**: 简洁、有力，适合商业提案

## 常见问题

### Q: 文件上传后一直显示处理中？

A: 文件处理需要一定时间，特别是大文件。请耐心等待，处理完成后状态会自动更新。

### Q: 搜索结果不准确？

A: 尝试：
- 使用更具体的搜索词
- 检查文档是否已完全处理
- 确认文档内容与搜索词相关

### Q: 对话回答不够准确？

A: RAG的回答质量取决于：
- 文档的完整性和质量
- 问题的清晰度
- 检索到的相关内容

### Q: 如何提高系统性能？

A: 建议：
- 定期清理不需要的文件
- 使用SSD存储向量数据库
- 增加系统内存

### Q: 数据安全如何保障？

A: 本系统完全本地运行：
- 所有数据存储在本地
- 不会上传到云端
- 您完全掌控数据
